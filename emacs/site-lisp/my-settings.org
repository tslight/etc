Please see the [[file:README.org][README]] and/or [[file:my-tangles.org][my-tangles.org]] for instructions on how to install.

As the name might suggest, this file mainly toggles various built in Emacs
variables to what I consider to be slightly saner defaults...

* EDITING SETTINGS

  (electric-indent-mode 1)
  (global-subword-mode 1) ;; move by camel case, etc
  (global-auto-revert-mode 1) ;; reload if file changed on disk

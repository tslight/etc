* PYTHON

Make sure the following python packages are installed with ~pip install~:

- *autopep8* /pep8 style formatter/
- *black* /uncompromising formatter/
- *flake8* /style guide enforcement/
- *importmagic* /automatic imports/
- *jedi* /lightweight autocomplete/
- *jupyter* /ipython successor/
- *rope* /refactoring library/
- *virtualenv* /sandboxed python environments/
- *yapf* /yet another python formatter/

#+BEGIN_SRC sh :tangle no
pip3 install autopep8 black flake8 importmagic jedi jupyter rope virtualenv yapf pylint
#+END_SRC

[[https://emacs.stackexchange.com/questions/2745/make-fill-paragraph-in-python-docstring-leave-the-triple-quotes-on-separate-line/2758][Docstring Fill Column Mode]]

#+BEGIN_SRC emacs-lisp
  (use-package python-mode
    :ensure t
    :commands (python-mode)
    :mode (("\\.py\\'" . python-mode))
    :config
    (setq fill-column 79)
    (setq python-shell-interpreter "ipython3")
    (setq python-fill-docstring-style 'django)
    ;; stop python interpreter from clobbering find-file...
    ;; https://answers.launchpad.net/python-mode/+question/221144
    (add-hook 'python-mode-hook
              (lambda ()
                ;; remove python-mode's ffap things that slow down find-file
                (setq ffap-alist (remove '(python-mode . py-ffap-module-path) ffap-alist))
                (setq ffap-alist (remove '(python-mode . py-module-path) ffap-alist))
                (setq ffap-alist (remove '(inferior-python-mode . py-ffap-module-path) ffap-alist))
                )))
#+END_SRC

** ELPY

https://elpy.readthedocs.io/en/latest/

Run ~elpy-config~ to check python dependencies

#+BEGIN_SRC emacs-lisp
  (use-package elpy
    :ensure t
    :defer t
    :hook
    (python-mode . elpy-mode)
    :config
    (elpy-enable)
    (setq elpy-rpc-backend "jedi")
    (setq python-shell-interpreter "jupyter"
          python-shell-interpreter-args "console --simple-prompt"
          python-shell-prompt-detect-failure-warning nil)
    (add-to-list 'python-shell-completion-native-disabled-interpreters"jupyter"))
#+END_SRC

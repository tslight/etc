* General

If running on a resolution sharper than 2000 horizontal pixels leave
more room for pixelclock

#+begin_src conf :tangle (if (> (string-to-number (shell-command-to-string "xdpyinfo | grep dimensions | awk '{print $2}' | awk -Fx '{print $1}'")) 2000) "~/.cwmrc" "no")
  gap 0 0 2 15
#+end_src

If less than 2000 pixels, 5 should be fine

#+begin_src conf :tangle (if (< (string-to-number (shell-command-to-string "xdpyinfo | grep dimensions | awk '{print $2}' | awk -Fx '{print $1}'")) 2000) "~/.cwmrc" "no")
  gap 0 0 2 5
#+end_src

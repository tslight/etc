* PRE-REQUISITES

Get Quicklisp from [[https://www.quicklisp.org/beta/][here,]] and then load into the REPL with:

 ~$ sbcl --load quicklisp.lisp~

From the REPL, install it:

#+BEGIN_SRC common-lisp :tangle no
  (quicklisp-quickstart:install)
#+END_SRC

And make sure you have added it to your lisp init file using:

#+BEGIN_SRC common-lisp :tangle no
  (ql:add-to-init-file)
#+END_SRC

Install stumpwm and some packages we need for niceties.

#+BEGIN_SRC common-lisp :tangle no
  (ql:quickload :stumpwm)      ;; Install stumpwm
  (ql:quickload :xembed)       ;; Required by stumptray
  (ql:quickload :clx-truetype) ;; Required by ttf-fonts
  (ql:quickload :swank)        ;; Required by slime-connect
#+END_SRC

Finally, clone the [[https://github.com/tslight/stumpwm-contrib][stumpwm-contrib]] repository to ~$HOME/src~, to be able use
ttf-fonts, cycle-mru and stumptray.

* Your hacking starts... NOW!

Stolen from [[https://github.com/alezost/stumpwm-config/blob/master/init.lisp][here]]

#+BEGIN_SRC common-lisp
  (in-package :stumpwm)
  (load "~/quicklisp/setup.lisp")

  (set-module-dir
   (pathname-as-directory
    (concat (getenv "HOME") "/src/stumpwm-contrib")))

  (defvar my/init-directory
    (directory-namestring
     (truename (merge-pathnames (user-homedir-pathname) ".stumpwm.d")))
    "Path to configuration directory.")

  (defun my/load (filename)
    "Load a file FILENAME (without extension) from `my/init-directory'."
    (let ((file (merge-pathnames (concat filename ".lisp")
                                 my/init-directory)))
      (if (probe-file file)
          (load file)
          (format *error-output* "File '~a' doesn't exist." file))))

  (redirect-all-output (merge-pathnames "log" my/init-directory))

  (my/load "settings")
  (my/load "utils")
  (my/load "keys")

  (setf *startup-message* (concat
                           "^B^5Welcome to StumpWM~%^n"
                           (random-encouragement)))
#+END_SRC

* AUTOSTART

After loading all our lisp, run some shell programs to make things more comfy.

#+BEGIN_SRC common-lisp
  (run-shell-command "xmodmap -e 'keycode 113 = Super_R'")
  (run-shell-command "xsetroot -cursor_name left_ptr")
  (run-shell-command "xfsettingsd")
  (run-shell-command "xsetroot -solid grey9")
  (run-shell-command "nm-applet")
  (run-shell-command "xfce4-power-manager")
  (run-shell-command "xfce4-clipman")
#+END_SRC
